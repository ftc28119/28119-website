# FTC 28119 & 30222 - 机器人竞赛团队网站

## 网站简介

这是FTC（First Tech Challenge）机器人竞赛团队28119 Infinite Dream和30222 Navigators的官方网站，托管于[28119.netlify.app](https://28119.netlify.app)。

## 团队信息

- **FTC 28119 Infinite Dream**
- **FTC 30222 Navigators**
- **所在地**：中国北京
- **当前赛季**：2025-2026赛季 (Decode)

## 网站内容

### 首页
展示团队名称、标志和机器人照片，提供快速导航至其他页面的入口。

### 关于我们
- 团队构成：由18名中学生和2名指导老师组成
- 参赛展望：积极备战2025-2026赛季，追求卓越

### 队伍使命
1. **提升技术能力**：深入探索机器人技术，提升机械设计、电子工程和编程能力
2. **取得优异成绩**：在比赛中取得佳绩，争取进入更高水平赛事
3. **培养团队精神**：培养创新思维、问题解决能力和团队协作精神
4. **推广STEM教育**：积极参与社区活动，推广STEM教育理念

### 核心价值
遵循FIRST的核心价值观，团队秉持：
- 创新：不断探索新技术，挑战自我
- 合作：尊重成员贡献，发挥集体智慧
- 学习：保持求知欲，不断提升自我
- 卓越：追求卓越，精益求精
- 包容：接纳不同背景和观点
- 乐趣：享受科技探索过程，保持热情
